# RNAmaps
An R package for creating sophisticated plots of genomic interval intersections.

## Dependencies
* R >= 3.3
 * dplyr
 * stringr
 * ggplot2
 * reshape2
 * smoother
* Python >= 2.7
 * pybedtools
 * csv
 * optparse
* bedtools >= 2.26

## Installation

devtools::install_github("jackhump/RNAmaps")
